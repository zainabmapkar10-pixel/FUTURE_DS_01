# FUTURE_DS_01
Business Sales Performance Dashboard including data cleaning, KPI analysis, dashboard creation and business recommendations using Power BI.

# Business Sales Performance Dashboard

# Project Overview
This project analyzes business sales data to identify key trends, profitable products and areas for improvement.

# Objectives
- Analyze revenue trends over time
- Identify top-performing products
- Evaluate category and regional performance
- Provide business recommendations

# Dataset
The dataset used in this project is the Superstore Sales Dataset, which contains business sales transaction data.

Source: Kaggle - SuperStore Sales Dataset

It includes:
- Product details (name, category, sub-category)
- Sales and profit values
- Customer segments (Consumer, Corporate, Home Office)
- Regional data (region, state)
- Order dates and shipping details

This dataset is suitable for analyzing sales performance, profitability and business trends.

# Data Cleaning & Preparation
Before analysis, the dataset was cleaned and prepared to ensure accuracy and consistency.

Steps performed:
- Removed missing and duplicate values
- Corrected data types (dates, numeric fields)
- Standardized category and product names
- Handled incorrect entries
- Created calculated columns/measures (Profit, Profit Margin, etc.)
- Verified data for analysis readiness

# Tools Used
- Power BI
- Excel

# Key Insights
- Total Revenue: 13M
- Total Profit: 1.47M
- Total Orders: 51K
- Profit Margin: 11.6%
- Avg Order Value: 246.50
- Technology category generates highest sales
- High discounts reduce profitability
- Some products cause significant losses

# Recommendations
- Reduce high discounts
- Focus on best-selling products
- Improve or remove loss-making products
- Increase average order value
